# surl.js

simple vdom implementation.

- safe templates, 
- opt in/out render loop, 
- animation helper,
- router
- components
- hyperscript helper
- fast.

source code self documented, see index.html. 3kb min+gziped, 6.8kb min. 

###Supports: Chrome, Firefox, Safari, IE9+ (for now router is only IE10+)
