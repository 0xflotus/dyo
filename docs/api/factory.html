	type: {...}
): object

setHost

Called when mounting to a host container. Use this to setup references that you can retrieve in getHost for the update phase.


	setHost: (element: Element, node: object) => void

setNode

Called when assigning an elements node. Use this to set references to instances created in createElement, createText and createEmpty


	setNode: (element: Element, node: object) => void

setContent

Called when initialiazing a mount container. Use this to setup the native view for a mount.


	setContent: (element: Element, children: Element) => void

setText

Called when updating a text elements value.


	setText: (element: Element, value: string) => void

setEvent

Called when assigning an elements event listeners. Use this to setup native event listeners. When an event is dispatched call Element.handleEvent with an Event object. The type of event type is the only required property in an Event object.


	setEvent: (element: Element, type: string) => void

setProps

Called when setting an elements individual properties. Use this to update or initialize properties.


	setProps: (element: Element, name: string, value: any, xmlns: string) => void
