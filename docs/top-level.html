Top-Level API

render

The entry point of every application starts with the render function.


render(element, container, callback?: function): thenable

createElement

Creates virtual elements that Dyo understands.


createElement(type, props?: object, ...children?: Array): object

cloneElement

Properties are shallow merged while new children replace old.


cloneElement(element, props?: object, ...children?: Array): object

isValidElement

Validates that the argument is a valid virtual element.


isValidElement(element: any): boolean

createPortal

Creates a virtual portal element that Dyo understands.


createPortal(children, container: string|object, props?: object): object

Component

The following is the base Component interface. This class is meant to be extended.


class Component {
	constructor (props: object)
	forceUpdate(callback?: function)
	setState(state?: object, callback?: function)
	render(props: object, state: object, context: object)
}

PureComponent

In a similar fashion the base PureComponent interface is identical to the Component interface. This interface is meant to be extended.


class PureComponent {
	constructor (props: object)
	forceUpdate(callback?: function)
	setState(state?: object, callback?: function)
	shouldComponentUpdate(props: object, state: object, context: object)
	render(props: object, state: object, context: object)
}

Children

The Children interface provides usefull helpers to use with the opaque props.children data structure.

toArray

Converts children to an array.


toArray(children: any): Array

forEach

Iterates through all the children, calling the provided callback with each child.


forEach(children: any, callback: function)

map

Iterates through all the children, calling the provided callback with each child, the result of which is added to the returned array.


map(children: any, callback: function): Array

only

Throws if children is not a single child element, returns the element otherwise.


only(children: element): object

count

Returns a count of the number of children.


count(children: any): number

filter

Filters through children using the provided callback as a comparator. The result is an array of children that pass the test.


filter(children: any, callback: function): Array

find

Filters through children to find the first match signaled by the provided comparator callback.


find(children: any, callback: function): any
