A Javascript Library
For Building User Interfaces

Declarative

DIO makes it painless to create interactive UIs. Design simple views for each state in your application, and DIO will efficiently update and render just the right components when your data changes.

Declarative views make your code more predictable and easier to reason about.

Component-Based

Build encapsulated components that manage their own state, composed to make complex UI's.
